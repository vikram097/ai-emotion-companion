# AI Emotion Companion

AI Emotion Companion is an intelligent system that understands user emotions and provides supportive responses using advanced AI techniques.

## Problem
Many people experience loneliness, stress, and emotional difficulties but do not always have someone to talk to.

## Solution
This project builds an AI-powered emotional companion that:
- Detects user emotions
- Provides empathetic responses
- Suggests helpful coping strategies

## Features
- Emotion detection using NLP
- Conversational AI chatbot
- Retrieval Augmented Generation (RAG)
- Personalized emotional support
- Memory-based conversation

## Tech Stack
- Python
- NLP
- Machine Learning
- LLMs
- Vector Database
- React (Frontend)
- FastAPI (Backend)

## Architecture
User → Frontend → Backend API → Emotion Detection → RAG System → LLM Response
